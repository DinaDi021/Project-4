Дейнерис Бурерожденная из дома Таргариенов

От крови древней Валирии, первая этого имени, королева андалов, ройнаров и Первых Людей,
владычица семи королевств, и хранительница областей, неопалимая, правительница Миэрина,
кхалиси Великого травяного моря, Разрушительница Оков и Матерь Драконов

Дейнерис — единственная дочь и младший ребёнок Эйриса II Таргариена и его сестры-жены Рейлы.
В ночь, когда я родилась, разразился сильный шторм, поэтому меня называют Бурерождённой.
Я — одна из важнейших персонажей серии книг, хотя мои главы географически оторваны от всех остальных:
на протяжении первых пяти книг я путешествует по странам Востока, пытаясь найти себе союзников,
вернуться в Семь Королевств и отвоевать Железный Трон.
Моей главной силой является обладание драконами, вернее скзаать - мои детки:

  1. Дрогон
  2. Визерион
  3. Рейегаль

Базовые команды гита

  1. git clone адрес репозитория - клонировать репозиторий на локальный компютер
  2. git commit -m 'initial commit' комит изменений в локальный репозиторий
  3. git push -u origin master - только первый раз отправка изменений в удаленный репозиторий
  4. git push - все последующие разы отправка изменений в удаленный репозиторий
